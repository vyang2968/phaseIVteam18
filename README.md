# phaseIVteam18

## Tech Stack
### Frontend
- Build Tool: Vite
- Package Manager: NPM
- Framework: React
- UI Library: Material UI

### Backend
- Flask

### Database
- NeonDB (PostgreSQL)

## Installation & Setup
### Frontend
-  `npm install`

### Backend
- Create / Activate your virtual enviroment
- `pip install -r requirements.txt`

## Running
### Frontend

### Backend
- `python run.py` (make sure you're in server folder)
